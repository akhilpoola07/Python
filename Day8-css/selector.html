1. Universal selector: Selects all elements on the page. It is represented by an asterisk (*). 2. Type selector: Selects all elements of a specific type. For example, p selects all paragraph elements.

3. Class selector: Selects all elements with a specific class attribute. It is represented by a period (.). For example, .class1 selects all elements with the class

4. ID selector: Selects a single element with a specific ID attribute. It is represented by a hash symbol (#). For example, #id1 selects the element with the ID id1.
